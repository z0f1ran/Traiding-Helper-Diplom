# Crypto Trading Platform

## Описание проекта
Веб-приложение для учебного моделирования криптоторговли. Приложение позволяет просматривать рыночные инструменты, анализировать портфель, создавать ордера на покупку/продажу и отслеживать историю операций.

## Цели и задачи
- реализовать многоэкранный пользовательский интерфейс;
- связать UI со слоем бизнес-логики и слоем данных;
- соблюсти архитектуру MVC и разделение ответственности между слоями;
- подготовить проект к финальной сдаче и защите ВКР.

## Технологический стек
- Backend: Python, Flask, Flask-SQLAlchemy
- Frontend: React (react-scripts)
- База данных: SQLite (по умолчанию), возможна замена через `DATABASE_URL`
- Контроль версий: Git, GitHub

## Архитектура приложения
Используется архитектура **MVC**.

### Backend
- `backend/app/models` — сущности данных (`Asset`, `Position`, `Order`)
- `backend/app/controllers` — бизнес-логика и слой доступа к данным (repositories + `TradingController`)
- `backend/app/views` — API-слой (Flask Blueprint, JSON-ответы)
- `backend/app/extensions.py` — инициализация `SQLAlchemy`
- `backend/app/__init__.py` — фабрика приложения `create_app`

### Frontend
- `frontend/src/App.js` — экраны и пользовательские сценарии
- `frontend/src/index.js` — точка входа React
- `frontend/src/App.css` — базовая стилизация интерфейса

## Пользовательские сценарии
- пользователь открывает дашборд и получает сводку по системе;
- переходит в раздел рынка и просматривает доступные инструменты;
- создаёт ордер на покупку/продажу;
- проверяет изменение портфеля после операции;
- анализирует историю созданных ордеров.

## Инструкция по запуску

### Backend
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python run.py
```

### Frontend
```bash
cd frontend
npm install
npm start
```

Frontend development server runs on `http://localhost:3001`.
